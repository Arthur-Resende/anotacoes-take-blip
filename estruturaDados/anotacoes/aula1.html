O que é Estrutura de dados

Por que aprender estrutura de dados?

Os computadores utilizam certas estruturas para armazenar dados em sua memória. Conhecer essas estruturas pode garantir que o programa tenha mais qualidade, visto que o programador faz o programa de forma mais otimizada para o sistema.

É possíveo que você também tenha mais liberdade em fazer programas. Visto que ele vê possibilidades antes não aparentes.

A morphcat games deu um ótimo exemplo mostrando como conhecer as estruturas utilizadas em um sistema permitem a criação de programas incríveis.

O que é estrutura de dados?

Uma estrutura organizada para armazenar dados na memória do computador ou qualquer dispositivo, com o objetivo de utilizar e manipular os dados de forma correta.

Existem estruturas especializadas para realizar o uso dos dados da forma mais eficiente para certa tarefa como estruturas de busca por exemplo.

Existem também estruturas mais generalistas, versáteis. Creio que Arrays costumam ser utilizados em diversos tipos de programas.

Como utilizar os dados?

Utilizando algoritmos, são instruções ordenadas para realizar uma tarefa. Podem ser usados para acessar, manipular ou utilizar dados guardados na memória em diversos tipos de estrutura.

Algumas operações essenciais são necesárias independentemente da estrutura utilizada, são elas:

  • Inserir e remover dados
  • Acessar todos os dados e a pesquisar um dado específico
  • Como categorizar os dados
